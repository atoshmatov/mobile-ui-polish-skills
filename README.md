# Mobile UI Polish Skills

Claude Code skills for making mobile interfaces feel polished and alive.
Works with Android (Compose + XML) and iOS (SwiftUI + UIKit).

## Skills

| Skill | Platform | Description |
|-------|----------|-------------|
| `compose-ui-polish` | Android — Jetpack Compose | Typography, spring animations, stagger, shadows, concentric corners, haptics |
| `android-xml-ui-polish` | Android — XML Views | TextAppearance, StateListDrawable, ObjectAnimator, TransitionManager, ShapeableImageView |
| `swiftui-ui-polish` | iOS — SwiftUI | ButtonStyle, spring, matchedGeometryEffect, stagger, SF Symbols animations |
| `uikit-ui-polish` | iOS — UIKit | CALayer, UIViewPropertyAnimator, CAGradientLayer, UIVisualEffectView, stagger |

## Install

```bash
npx skills add alimardon/mobile-ui-polish-skills
```

Or manually copy skill folders to `~/.claude/skills/`.

## What's Covered (All Platforms)

- **Typography** — letterSpacing, lineHeight, custom fonts
- **Press states** — scale on tap, spring physics
- **Enter/Exit transitions** — fade, slide, scale
- **Stagger animations** — list items one by one
- **Shadows** — ambient + spot shadow
- **Concentric corner radius** — inner = outer − padding
- **Image polish** — clip, border, gradient overlay, shimmer
- **Haptic feedback**
- **Loading skeletons / shimmer**
- **Optical alignment**
- **Spacing rhythm** — 4/8/16/24/32dp
